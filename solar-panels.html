Günəş Paneli Sistemləri

Eviniz və ya biznesiniz üçün ən uyğun günəş enerjisi həllini birlikdə tapaq.

On-Grid Sistem

Elektrik şəbəkəsinə birbaşa qoşulma. Artıq enerjini şəbəkəyə geri satış imkanı. Ən sərfəli investisiya.

✓ Şəbəkəyə geri satış
✓ Ən aşağı qiymət
✓ Sürətli geri dönüş
✗ Elektrik kəsilməsində işləmir

Hibrid Sistem

On-Grid və Off-Grid-in birləşməsi. Həm şəbəkə, həm batareya dəstəyi — maksimum etibarlılıq.

✓ Şəbəkə + batareya dəstəyi
✓ Elektrik kəsilməsində işləyir
✓ Geri satış mümkündür
✓ Ən etibarlı həll
Hesabla

Off-Grid Sistem

Şəbəkəsiz ərazilərdə tam avtonom. Günəş paneli + akkumlyator + invertor. Gecə boyunca fasiləsiz enerji.

✓ Tam müstəqillik
✓ Fasiləsiz enerji
✓ Uzaq ərazilər üçün ideal
✗ Daha yüksək başlanğıc xərc
